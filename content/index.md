# What is Peace Fixation?

This is my personal website for sharing things that are interesting to me. The name Peace Fixation comes from an old psychedelic techno [record](https://www.discogs.com/master/546309-Freaky-Chakra-Peace-Fixation) by an artist called Freaky Chakra and I like the sentiment.

I've designed the website to store deeply nested lists so I can categorize things in a way that makes sense to me. It's mostly a way to organize my bookmarks. Perhaps you'll find something interesting here too.
